## 공급유형(supplyType) 문서 근거 (data.go.kr 1차 출처, 2026-06)

`supplyType`은 닫힌 단일 코드표가 없다. 소스별 문서화 수준이 다르다.

- **LH 분양임대공고문(15058530, 우리가 수집)**: 활용가이드에 **공고유형 `UPP_AIS_TP_CD` 6개만** 코드표로 명시 — `01 토지·05 분양주택·06 임대주택·13 주거복지·22 상가·39 신혼희망타운`. 세부유형 `AIS_TP_CD_NM`은 응답 출력 필드로 예시(행복주택)만 있고 코드표 없음. (지역코드 17·공고상태코드 5개: 공고중/접수중/접수마감/상담요청/정정공고중도 명시)
- **LH 공급정보조회(15056765)**: 설명에 세부 매물유형 **31종** 나열(공공임대·국민임대·영구임대·행복주택·통합공공임대·각종 매입/전세임대·토지·상가 등). 정확한 코드↔명 표는 자체 활용가이드 docx.
